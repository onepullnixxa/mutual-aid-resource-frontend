# Mutual Aid Resource

A full-stack React application which boosts aid requests for Black trans and nonbinary individuals in need of assistance.

## Technologies Used

- HTML 5
- CSS3
- JavaScript
- JSX
- React
- Express.js
- Node.js
- Mongoose
- MongoDB
